## **Install instructions**

### Getting Started

#### 1) Clone & Install Dependencies

- 1.1) `git clone`
- 1.2) `cd artia-ui`
- 1.3) Install NPM packages with `npm install` or `yarn install`
   
#### 2) Launch the application

- 2.1) Run `npm start` or `yarn start` in your terminal
